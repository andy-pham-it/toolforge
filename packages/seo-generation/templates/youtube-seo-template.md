# YouTube SEO Template

## Title
```
{Primary Keyword} - {Secondary Keyword} | {Episode/Topic Description}
```

**Rules:**
- Primary keyword ở đầu title (60-70 ký tự tối đa)
- Dùng dấu `|` hoặc `-` để phân cách
- Include số (list, year, number) nếu có thể
- Không dùng clickbait sai lệch

**Examples:**
- ✅ "React Tutorial for Beginners 2026 - Build Your First App"
- ❌ "React Tutorial" (quá ngắn, thiếu keyword)

---

## Description

```
{2-3 câu đầu chứa primary keyword, mô tả nội dung tập}

📌 Nội dung chính:
00:00 - Giới thiệu
XX:XX - Phần 1: {Nội dung}
XX:XX - Phần 2: {Nội dung}
XX:XX - Kết luận

🔗 Tài nguyên:
- Link 1
- Link 2

#hashtag1 #hashtag2 #hashtag3
```

**Rules:**
- 200-500 ký tự
- Keyword trong 2 dòng đầu
- Timestamps cho video > 5 phút
- 3-5 hashtags
- Link tài nguyên ở cuối

---

## Tags

```
[{keyword1}, {keyword2}, {keyword3}, {keyword4}, {keyword5}]
```

- 3-5 tags
- Tag đầu là primary keyword
- Include long-tail variations
