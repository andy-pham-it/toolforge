# Business Analysis Support Hub

## MCP Tools
All tools via: `skill_mcp(mcp_name="andy-toolforge", tool_name="<name>")`

| Tool | Description |
|------|-------------|
| `toolforge_competitor_analysis` | Crawl website + profile competitor business |
| `toolforge_pricing_analysis` | Analyze competitor pricing data → strategic insights |
| `toolforge_swot_analysis` | Generate SWOT from competitor profiles |
| `toolforge_trend_analysis` | Analyze market trends by keywords |
| `toolforge_business_report` | Generate comprehensive business analysis report |

## Workflow
1. `competitor_analysis` — profile each competitor
2. `pricing_analysis` — analyze pricing data
3. `swot_analysis` — synthesize findings into SWOT
4. `trend_analysis` — research market trends
5. `business_report` — compile final report

## Related Skills
- `toolforge-content-research-hub` — competitor analyzer for content
- `toolforge-podcast-content-strategy` — content gap analysis

## Integration
- BA research feeds into content strategy (what to cover, positioning)
- Market trends inform content ideation
- SWOT/research used for strategic planning docs
