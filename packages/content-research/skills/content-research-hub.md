# Content Research Hub

## MCP Tools
All tools via: `skill_mcp(mcp_name="andy-toolforge", tool_name="<name>")`

| Tool | Description |
|------|-------------|
| `andy_toolforge_content_summarizer` | Summarize articles/reports via LLM with skill-file prompts |
| `andy_toolforge_content_ideator` | Generate content ideas by topic, audience, format |
| `andy_toolforge_article_manager` | Article lifecycle: classify, tag, summarize, improve |
| `andy_toolforge_competitor_analyzer` | Crawl competitor URL + SWOT analysis via LLM |

## Workflow
1. `content_ideator` — brainstorm topics
2. `content_summarizer` — research existing content
3. `competitor_analyzer` — analyze competitor content
4. `article_manager` — classify, tag, and improve articles

## Related Skills
- `toolforge-podcast-content-strategy` — research + SEO workflow
- `toolforge-ba-support-hub` — deeper business/market analysis
- `toolforge-content-operations-hub` — content planning + distribution

## Integration
- Research outputs feed into content-operations for planning
- Summaries used as source material for scripts
- Competitor analysis informs SEO strategy via seo-generation
